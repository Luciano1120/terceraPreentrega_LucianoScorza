## Mi Pagina Web contiene 4 Modelos

1. Items
2. Clientes
3. Proveedores
4. Avatar

El CRUD se hizo con el Modelo Proveedor. 
Permitiendo Agregar un Proveedor, leer los Proveedores Cargados, Modificar y Eliminar cada uno de ellos.
